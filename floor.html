#include <math.h> floor(float)

Explanation:

Description: The floor function calculates the integer no larger than the argument.
Parameters: float
Returns: Integer

Example:

/*floor example */
#include <stdio.h> /* printf */
#include <math.h> /* floor */

int main ()
{
printf ("The floor of 17.6 is: ", floor(17.6));
return 0;
}

Output:

The floor of 17.6 is: 17
